Direct VAPI Assistant Integration Demo

About This Demo

This page demonstrates how to use a VAPI assistant ID directly with the voice bot widget, bypassing the need for a bot record in the database.

This approach is useful when:

  • You've created an assistant directly in the VAPI dashboard
  • You're experiencing issues with the bot database
  • You want to quickly test a VAPI assistant without creating a bot

Integration Code

The following script is embedded in this page:

<script defer src="/js/external-chatbot-voice.js" data-vapi-assistant-id="YOUR_VAPI_ASSISTANT_ID" data-language="en" data-position="right" data-theme="light"></script>

Important: Replace YOUR_VAPI_ASSISTANT_ID with your actual VAPI assistant ID from the VAPI dashboard.

You can find your assistant ID in the VAPI dashboard under the Assistants section.

How It Works

When the data-vapi-assistant-id attribute is provided:

  1. The script uses this ID directly without checking the bot status API
  2. The voice bot is activated immediately
  3. All voice interactions are handled by the specified VAPI assistant

This approach simplifies integration when you already have a VAPI assistant set up.
